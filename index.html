Not So Fake

Este projeto foi desenvolvido a fim de criar uma aplicação web com o objetivo de combater os malefícios causados pela disseminação de notícias falsas. As principais ferramentas disponíveis são: Blog estático informativo sobre as Fake News, Feed RSS que reúne notícias de fontes específicas e Buscador de notícias disponibilizado pelo Google API com filtro de resultados de fontes específicas pré definidas.

Foi desenvolvido através das técnologias HTML, CSS e JavaScript, além da biblioteca Front-End Bootstrap.

ToDo

Este projeto foi desenvolvido a fim de criar uma urna eletrônica para votação. O funcionamento desta urna ocorre através de ferramentas de inserção de votos pelo usuário, contagem dos votos e armazenamento a longo prazo dos dados através do LocalStorage do navegador do usuário. A sessão de votação pode ser finalizada e os dados resetados a qualquer momento pelo usuário administrador.

Foram utilizadas as técnologias Front-End: HTML, CSS e JavaScript. Também foi utilizado a biblioteca Bootstrap.
